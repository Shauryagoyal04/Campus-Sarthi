Campus SARTHI Widget Demo

Embeddable multilingual chatbot for campus support

Quick Start

Add the following code to your website to embed the Campus SARTHI widget:

<!-- Load the widget script -->
<script src="https://your-domain.com/widget-loader.js"></script>

<!-- Initialize the widget -->
<script>
  CampusSarthi.init({
    host: 'https://your-domain.com',
    apiKey: 'your-api-key',
    defaultLang: 'en' // en, pa, te, bn
  });
</script>

Configuration Options

CampusSarthi.init({
  host: 'https://your-domain.com',  // Required: Your widget host
  apiKey: 'your-api-key',        // Required: Your API key
  elId: 'custom-widget-id',       // Optional: Custom element ID
  defaultLang: 'en',              // Optional: en, pa, te, bn
  theme: 'light',                 // Optional: light, dark
  position: 'bottom-right'         // Optional: bottom-right, bottom-left, etc.
});

API Methods

// Destroy a widget instance
CampusSarthi.destroy('widget-id');

// Get all active instances
const instances = CampusSarthi.getInstances();

// Update widget configuration
CampusSarthi.updateConfig('widget-id', { defaultLang: 'pa' });

// Check version
console.log(CampusSarthi.version);

Events

Listen to custom events from the widget:

// Listen for widget events
window.addEventListener('campusSarthi:messageReceived', (event) => {
  console.log('Message received:', event.detail);
});

window.addEventListener('campusSarthi:escalated', (event) => {
  console.log('Escalated to human:', event.detail);
});
